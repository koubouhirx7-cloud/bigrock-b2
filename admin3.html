01. PROJECT OVERVIEW

A high-performance B2B procurement hub designed for Japan's premier mountain bike distributor. The interface follows an "Industrial Precision" aesthetic, mirroring the rugged, technical nature of CNC-machined MTB components. It prioritizes data density, visual stock verification, and speed-optimized workflows for dealers and workshop managers.

02. TECH STACK REQUIREMENTS

  • ›
    Frontend Next.js (App Router), Tailwind CSS v3, Lucide React icons.
  • ›
    Authentication NextAuth.js with Google Provider (Enterprise OAuth).
  • ›
    Backend/DB Supabase (PostgreSQL) with Row Level Security.
