01. AUTH

Authentication & Session Management
GET /auth/google
Internal / External

Redirects user to Google OAuth flow for authentication.

RESPONSE: 302 REDIRECT
// Header: Location: https://accounts.google.com/o/oauth2/v2/...
GET /auth/session

Validates current session and returns active user context.

{
  "status": "success",
  "data": {
    "user_id": "usr_9921",
    "role": "procurement_manager",
    "expires_at": "2023-12-31T23:59:59Z"
  }
}

02. PRODUCTS

Catalog & Pricing
GET /products

Retrieve list of products with optional query parameters: category, manufacturer, and search.

{
  "count": 1204,
  "results": [
    { "id": "BR-101", "name": "Heavy Duty Winch", "stock": 45 },
    { "id": "BR-102", "name": "Titanium Alloy Bolt", "stock": 1200 }
  ]
}
GET /products/{id}

Retrieve full product details including technical specs and volume-based price tiers.

{
  "id": "BR-101",
  "price_tiers": [
    { "min_qty": 1, "price": 450.00 },
    { "min_qty": 10, "price": 415.00 },
    { "min_qty": 50, "price": 380.00 }
  ]
}

03. ORDERS

Order Management & Lifecycle
POST /orders

Create a new procurement order. Validates against current budget limits.

// REQUEST BODY
{
  "items": [{ "product_id": "BR-101", "quantity": 5 }],
  "shipping_id": "addr_99"
}
POST /orders/{id}/reorder

Duplicate a previous order with current pricing and availability.

04. IMPORT

Bulk Data Ingestion
POST /import/csv

Multipart file upload for bulk order creation or inventory updates.

NOTE: MAX FILE SIZE 50MB. SUPPORTED FORMATS: .CSV, .XLSX

05. BUDGET

Financial Controls
GET /budget/status

Current fiscal period usage and remaining balance.

PATCH /budget/config

Update budget thresholds and approval workflow triggers.
